# 🍔 FoodExpress

A modern and responsive food delivery website inspired by popular food ordering platforms. FoodExpress allows users to browse restaurants, explore menus, add items to a cart, and place orders through a clean and user-friendly interface.

---

## 🚀 Features

### 🏠 Home Page
- Attractive landing page
- Hero section with food carousel
- Popular food categories
- Featured restaurants
- Customer testimonials
- About Us section
- Responsive navigation bar

### 🍽 Restaurants Page
- Restaurant listing
- Search functionality
- Category filtering using jQuery
- Responsive restaurant cards

### 📋 Menu Page
- Display menu items
- Add items to cart
- Interactive user interface

### 🛒 Cart Page
- View selected items
- Remove items from cart
- Dynamic total calculation

### 💳 Checkout Page
- Customer information form
- Order confirmation functionality

### 📞 Contact Page
- Contact form
- User-friendly design

### 🌙 Additional Features
- Dark Mode
- Responsive Design
- Smooth Scrolling
- Scroll-to-Top Button
- Bootstrap Components
- Interactive JavaScript Features
